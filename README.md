The Hospital Management System is a database-driven application designed to manage hospital operations efficiently. 
It enables administrators, doctors, and staff to handle patient records, doctor details, and appointment scheduling 
in a centralized system.

The project uses MySQL as the database to store and manage data for patients, doctors, and appointments with proper
relationships and constraints. It ensures secure data handling, quick access to information, and smooth coordination
between different hospital departments.

Key Features:
Manage patient details (registration).
Manage doctor information (specialization, availability).
Appointment booking between patients and doctors.
View and update treatment records  details.
Role-based access for admin, doctor, and staff
